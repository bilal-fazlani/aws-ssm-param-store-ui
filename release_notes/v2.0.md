# v2.0

A major update that adds full support for parameter descriptions, type editing, and StringList parameters — plus quality-of-life improvements across the board.

## What's New

**Parameter Descriptions**
- View and edit the optional description field on any parameter
- Descriptions are visible in the sidebar and detail view

**Type Editing**
- Change a parameter's type between String, SecureString, and StringList directly from the detail view
- Type changes take effect immediately on save

**StringList Parameters**
- Full support for AWS SSM StringList parameters
- A dedicated list editor lets you add, remove, and reorder individual items
- StringList parameters are shown with a purple "List" badge in the sidebar

**Folders with Values**
- Folders that are also parameters (e.g. `/app/config` alongside `/app/config/db`) now display correctly
- Switch between viewing the folder's own value and its child parameters

**Keyboard Shortcuts**
- Press ⌘⌫ to delete the currently selected parameter or folder

**What's New Overlay**
- After updating, a "What's New" overlay shows the release notes for the latest version on first launch

**Usability Improvements**
- After adding a new parameter, the value editor is automatically focused so you can start typing immediately
- Adding a parameter with a duplicate path is now prevented with a clear error message
- Parameters with empty values are now loaded and displayed correctly
